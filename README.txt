Aplicação em C, usando a biblioteca Raylib.

Jogo chamado "ZeldaINF", baseado no primeiro jogo da franquia "A Lenda de Zelda", publicado pela Nintendo.
O jogador controla um personagem que é capaz de se movimentar pelo cenário e matar monstros. Ao matar todos os monstros do nível, o jogador avança para o próximo nível. Ao perder todas as vidas para os monstros, o jogador perde o jogo.
